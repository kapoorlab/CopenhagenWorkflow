üî¨ Pipeline Overview

Each step of the pipeline is a standalone script with its own configuration:

  • 00_create_nuclei_membrane_splits.py ‚Äì generate Merged.tif and split channels

  • 01_nuclei_segmentation.py ‚Äì StarDist 3D nuclear segmentation

  • 01_enhance_membrane.py ‚Äì CARE denoising for membrane channel

  • 01_vollcellpose_membrane_segmentation.py ‚Äì Cellpose 2D membrane segmentation and 3D reconstruction

  • 02_oneat_nuclei.py ‚Äì mitosis classification using Oneat

  • 03_nms_nuclei_automated.py ‚Äì non-max suppression (automated)

  • 03_nms_nuclei_interactive.py ‚Äì non-max suppression (interactive via Napari)

All parameters are controlled via YAML files in the conf/ directory.
